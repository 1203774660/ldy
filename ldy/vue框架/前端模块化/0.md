# 模块化相关规范
传统开发模式：命名冲突，文件以来---》模块化解决
模块化：把单独的一个功能封装到一个模块。模块之间相互隔离，可以通过特定接口公开内部成员，也可以依赖别的模块
--》方便代码重用，开发快，方便维护
